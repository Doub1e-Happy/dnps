# Do Not Put Shit in my workspace
---
本项目的目的是构建一个Ai的行为模式,规范地使用workspace,不要到处堆各种垃圾代码，不同用途的代码不要混在一起。

DNPS

|___main/   任务的发起入口，编写这个提示词发起任务，让他读这个代码执行

|___rules/  定义一些规则，从main读这个里面的部分文件

|___roles/  职责文件，规定AI的身份
